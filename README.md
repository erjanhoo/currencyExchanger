# Currency Converter

A modern, responsive currency converter web application with live exchange rates.

## Features

- **Live Exchange Rates** - Real-time currency conversion using ExchangeRate-API
- **40+ Currencies** - Support for major world currencies
- **Interactive UI** - Clean, modern interface with dropdown selectors
- **Currency Swap** - Quick swap between currencies
- **Flag Icons** - Visual representation with country flags
- **Responsive Design** - Works seamlessly on desktop and mobile devices

## Getting Started

1. Open `frontend/index.html` in your web browser
2. Enter the amount you want to convert
3. Select your source and target currencies
4. Click "Convert" to see the result

## Technologies Used

- HTML5
- CSS3 (with gradient backgrounds and modern styling)
- Vanilla JavaScript
- ExchangeRate-API for live rates
- Flagpedia for country flag images

## Usage

Simply open the `index.html` file in any modern web browser. No installation or build process required.


## API

This application uses [ExchangeRate-API](https://www.exchangerate-api.com/) for fetching live currency exchange rates.
